Diretiva resonsavél por criar um elemento label e um radio na tela recebe os parametros:

label: Exibe o label com o valor passado pelo componente.

model: Passa o model que onde o valor será selecionado.

data: Passa as opções que serão exibidas na diretiva.

space: define a classe que sera usada como espacamento entre os elemento,

text define a propriedade do objeto que sera exibida no radiobuttom (texto do objeto)

default define o valor padrao que sera iniciado a diretiva

campo define o campo do objeto que será atribuido ao model
